Krafty Manual

Head Checker

Report what can be checked automatically, preview how the page appears in search results and when shared, and list every value in the head.

Nest Checker

Highlight elements that their parent element is not allowed to contain, explain each one on hover, and count them by kind.

Heading Checker

Report what can be checked automatically about the heading structure, and list the headings as an indented outline for you to read.

Outline Checker

Display an outline around every element.

Alt Checker

Display the alt attribute of every image, and flag images whose alt is missing or empty.

Brightness Checker

Make the page monochrome.

Keyboard shortcuts

Every checker can be toggled from the keyboard, but none come with a key already assigned. Pick your own here:

chrome://extensions/shortcuts

Krafty ships no default keys on purpose. Chrome drops a suggested key that anything else already claims, and does so without an error anywhere, so a default that looks reasonable can simply never work and leave the feature looking broken. Choosing your own takes a moment and always works.
